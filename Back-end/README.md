# 后端：Flask开发
1. ### app.py:程序入口  在这里实现接口类
2. ### dbGondig.py: 数据库配置 指定数据库主机 账户 端口 数据库名
3. ### model.py: 数据库实体 每个表对应一个Class类
4. ### server.py：数据库服务  连接数据库  定义数据库的增删改查等基本操作规则
5. ### utils.py：工具类  定义生成md5编码 以及证书token生成等
